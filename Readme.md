# A sample app to showcase the Slim Kernel Testing Harness

[Slim Kernel Testing Harness][harness] is a library which helps setting up
high-level tests using Slim Framework. This is a sample API with some business
requirements, entirely tested using the library.

## How to run?

```
composer install
composer run cs
```

[harness]: https://github.com/WonderNetwork/slim-kernel-testing-harness
